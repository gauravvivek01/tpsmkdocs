    Installing pre-built binaries with vcpkg

    The package installer vcpkg can be used to install many of the dependent libraries. Assuming it is installed in c:\vcpkg\vcpkg, open a command prompt in that directory and run

    • vcpkg install cairo, from c:\vcpkg\vcpkg
    • vcpkg install ffmpeg, from c:\vcpkg\vcpkg
    • vcpkg install libusb, from c:\vcpkg\vcpkg

    How to configure cmake

    :: Create a directory for the dependencies
:: mkdir C:\Users\BBOX\SDKVM\deps
:: Copy all of the lib, bin and include directories into this dependencies directory from the dependent libraries you downloaded. for example from C:\vcpkg\vcpkg\packages
:: We can then specify the dependency path with: -DCMAKE_PREFIX_PATH. This tells cmake where to look for libraries on windows systems.


SET DEPENDENCIES=C:\Users\BBOX\SDKVM\deps
SET BASE_DIRECTORY=C:\Users\BBOX\SDKVM\FreeRDP
SET VCPKGS_DIRECTORY=C:\vcpkg\vcpkg\packages
SET BUILD_DIRECTORY=%BASE_DIRECTORY%\FreeRDP_Build
SET SOURCE_DIRECTORY=C:\Users\BBOX\SDKVM\FreeRDP\FreeRDP

:: Remove all generated CMake files
cd /d %BUILD_DIRECTORY%
for /F "delims=" %%i in ('dir /b') do (rmdir "%%i" /s/q || del "%%i" /s/q)
cd ..



cmake -GNinja -B%BUILD_DIRECTORY%  -H%SOURCE_DIRECTORY%  -DCMAKE_PREFIX_PATH=%DEPENDENCIES% -DCMAKE_BUILD_TYPE=Release -DCHANNEL_URBDRC=OFF -DWITH_CAIRO=ON -DWITH_SWSCALE=ON -DWITH_FFPMEG=ON
cmake --build %BUILD_DIRECTORY%

    How To Configure Include and Library Directories

    In windows, it is recommended that one use a single dependency directory as ashown above rather than set individual directories.

    SET OPENSSL_ROOT_DIR=C:\Program Files (x86)\OpenSSL-Win32
SET LIBUSB_1_LIBRARY=%VCPKGS_DIRECTORY%\libusb_x86-windows\lib\libusb-1.0.lib
SET LIBUSB_1_INCLUDE_DIR=%VCPKGS_DIRECTORY%\libusb_x86-windows\include\libusb-1.0
SET SWScale_INCLUDE_DIR=%VCPKGS_DIRECTORY%\ffmpeg_x86-windows\include\libswscale
SET SWScale_LIBRARY=%VCPKGS_DIRECTORY%\ffmpeg_x86-windows\lib\swscale.lib
SET AVCODEC_INCLUDE_DIR=%VCPKGS_DIRECTORY%\ffmpeg_x86-windows\include\libavcodec
SET AVCODEC_LIBRARY=%VCPKGS_DIRECTORY%\ffmpeg_x86-windows\lib
SET AVUTIL_LIBRARY=%VCPKGS_DIRECTORY%\ffmpeg_x86-windows\lib
SET AVUTIL_INCLUDE_DIR=%VCPKGS_DIRECTORY%\ffmpeg_x86-windows\include\libavutil
SET SWRESAMPLE_LIBRARY=%VCPKGS_DIRECTORY%\ffmpeg_x86-windows\lib
SET SWRESAMPLE_INCLUDE_DIR=%VCPKGS_DIRECTORY%\ffmpeg_x86-windows\include\libswresample

SET CAIRO_INCLUDE_DIR=%VCPKGS_DIRECTORY%\cairo_x86-windows\include\cairo
SET CAIRO_LIBRARY=%VCPKGS_DIRECTORY%\vcpkg\packages\cairo_x86-windows\lib

+compression (default:off) Compression
