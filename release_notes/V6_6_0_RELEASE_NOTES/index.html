    1.7 6.6.0_r11141 RC1

    This release is available in \10.211.128.208\tftpboot\work\Emerald_Releases Source branch: develop/

    1.7.1 New functionality added/updated

    • Inactivity Timers:
      • Ability to configured Inactivity timers on a managed receiver via the OSD added
      • Rest API added to allow Boxilla to configure inactivity timers on a per RX basis
    • Cutthrough
      • Enabled cutthrough for Lossless (4k) connections.
    • New Boards
      • Added support for DVI SE and PE R2 boards (New boards with net2272 replaced with Artix implementation)
    • Support for Touchscreens over shared connection via Absolute Mouse added. This is a POC for a scecific customer. It is turned OFF by default. It can be enabled over SSH.
